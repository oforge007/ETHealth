# ETHealth – Telemedicine for the Unbanked

A mobile-first telemedicine gateway that lets rural Ethiopian farmers and unbanked families pay for doctor consultations using only **MiniPay + cUSD** on Celo.

No bank account needed · 24/7 access · Pay per consult (as low as 3–10 cUSD)

## Live Demo (after deploy)
https://ethealth.vercel.app

## Vision
- Farmer feels sick → opens ETHealth in Opera Mini  
- Chooses symptom → books 15-min video call  
- Pays 5 cUSD via MiniPay → money locked in smart-contract escrow  
- Doctor finishes call → patient confirms → doctor instantly receives cUSD  

Built 100% on Celo → ready for Celo ZK transition.
